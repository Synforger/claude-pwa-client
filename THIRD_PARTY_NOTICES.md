## Weak copyleft 説明

**MPL-2.0** (= pywebpush / py-vapid / certifi / pathspec) は file-level weak copyleft。 該当 file 自体を改変した場合のみ改変版を MPL-2.0 で公開する義務があるが、 別 file から import / dynamic link する分には影響しない (= 本リポは import 利用のみで MPL ファイルを改変していない)。
